How to install Diamond Toolbox Add-On for DiamondHunt

As a bookmarklet

  1. Paste the following code into the URL section of a new bookmark, and name it "Diamond Toolbox" 
    javascript:(function(){document.body.appendChild(document.createElement('script')).src='https://unnecessary-axiom.github.io/diamond_toolbox/diamond_toolbox.js';})();
  2. Run the bookmark when DiamondHunt is loaded
  3. A Diamond Toolbox now loads on the page and addds it's control tab

Other methods

  • The bookmarket is the only method available at this time. A userscript is possible in the future, a browser extension is not likely.
